# Record Count Per User in Salesforce

This project provides code to retrieve the count of records created by each user, categorized by Salesforce object type. It is useful for analyzing user activity and understanding object-wise record creation trends within your Salesforce instance.

## Features

- Counts records created by each Salesforce user.
- Breaks down counts by individual Salesforce objects (e.g., Account, Contact, Opportunity).
- Helps admins and analysts understand user-level object creation patterns.

## Usage

- Deploy the code in your Salesforce environment (details in the following sections).
- Run the provided scripts or queries to generate user-object record counts.

## Prerequisites

- Access to Salesforce with appropriate permissions.
- API or platform tools as required by your implementation.
